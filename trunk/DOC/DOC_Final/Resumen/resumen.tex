\begin{abstract}

Este trabajo presenta la aplicación de técnicas de computación paralela usando una GPU (Graphic Processing Unit) para mejorar la eficiencia computacional del algoritmo de \emph{ray tracing}.

El trabajo presenta un relevamiento del estado del arte de los algoritmos de generación de imágenes foto-realistas por computadora, se estudian las diferentes alternativas disponibles y las posibilidades de paralelización de cada una. Se investigan las paralelizaciones existentes del algoritmo de \emph{ray tracing} tanto en clusters como en GPU. Así mismo se realiza un relevamiento de las diferentes estrategias para verificar la calidad de las imágenes generadas y se selecciona un conjunto de casos de prueba relevantes.

Se obtiene el diseño e implementación de tres versiones del algoritmo de \emph{ray tracing} con una performance comparable a las mejores soluciones que ejecutan en GPU, así como una implementación en CPU para realizar la comparación de performance. Para lograr esta performance se tuvo que evaluar alternativas de optimización y la utilización de estructuras de aceleración. Se eligen los casos de prueba y realiza una evaluación de los resultados obtenidos en las mismas.

Los resultados experimentales obtenidos a partir de las implementaciones propuestas demuestran que existe una significante reducción del tiempo de ejecución frente a implementaciones para CPU, marcando un importante avance hacia la generación de imágenes en tiempo real sobre computadoras de escritorio.

\Keywords{computación gráfica, GPU, GPGPU, paralelismo, \emph{ray tracing}.}

\end{abstract}
